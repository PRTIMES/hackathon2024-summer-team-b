# PR Career

チームB「いちご騎士団🍓」

## Get started

### 1. Clone this repository

```bash
gh repo clone PRTIMES/hackathon2024-summer-team-b
```

### 2. Install dependencies

```bash
corepack enable

pnpm install
```

### 3. Start development server

```bash
pnpm dev
```
